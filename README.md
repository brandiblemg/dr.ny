# dr.ny

Dr. Nader Youssef – Personal Brand Platform (Prototype)

## Project Overview

This project is the prototype homepage for Dr. Nader Youssef’s consulting and thought leadership platform.

This is not a typical healthcare website.
It is a framework-driven authority platform built around:

ACE Framework (Advocacy, Compassion, Education)

Empathetic Engagement Through Education (Triple E)

Human-centered healthcare reform

The homepage serves as the foundation for a future multi-page site and must be structured with scalability in mind.

Strategic Positioning

Dr. Youssef sits at the intersection of:

Patient advocacy

Pharmaceutical study design

Healthcare provider communication training

This website must feel:

Premium

Modern

Human

Intelligent

High-authority

Non-clinical

Non-template

The goal is to differentiate from traditional medical consulting websites and elevate the brand to keynote-level positioning.

Typography Direction

Primary font: Modern sans-serif (clean, editorial)

Optional accent serif for large headings

Strong hierarchy

Generous whitespace

No heavy or crowded layouts

Homepage Structure

The homepage must follow this section order:

Sticky Navigation

Hero Section (video-forward)

Choose Your Path (3 audience cards)

ACE Framework visualization

How It Works (3 vertical summaries)

Dr. Youssef in Action (media grid)

Thought Leadership Preview

Final CTA

Footer

All sections should be modular and reusable.

Technical Guidelines

Use clean, semantic HTML structure

Build with Tailwind CSS (CDN acceptable for prototype)

Avoid Canva-specific artifacts

Components must be reusable (cards, accordions, grid layouts)

No hard-coded inline styles unless necessary

Maintain consistent spacing scale

Use responsive-first approach

This prototype is for evaluation and strategic alignment.
It will likely evolve into a multi-page production site.

Interaction Expectations (Prototype Level)

Hover states on buttons and cards

Expandable elements for framework explanations

Scroll-based transitions (light, subtle)

Video placeholders with play overlay

No heavy animation libraries required yet
